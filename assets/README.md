# App Icons

Place these icon files here for the app installer:

- `icon.ico` — Windows icon (256x256, multi-resolution .ico file)
- `icon.icns` — macOS icon (1024x1024 .icns file)
- `icon.png` — Linux icon (512x512 .png file)

## How to generate:

1. Start with a 1024x1024 PNG of your SyncBooks logo
2. Use https://www.electron.build/icons or https://iconverticons.com/online/
3. Upload your PNG and download all formats

The icon should be your SyncBooks green "S" logo on a rounded square background.
